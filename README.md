📊 NIFTY 50 Stock Market Analysis Dashboard
📌 Project Overview

This project is an interactive Stock Market Analysis Dashboard developed using Power BI to analyze NIFTY 50 stock data (2000–2021).

The dashboard provides insights into:

Stock price trends
Company-wise returns
Market volatility
Trading activity
Top gainers and losers
